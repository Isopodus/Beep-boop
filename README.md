### What is it?
This is a team project for Int20h Hackatone 2020.

Website: https://beep-boop-music.herokuapp.com/

#### Clone repository
`git clone https://github.com/Isopodus/Beep-boop`

#### Install dependencies
`yarn`
or
`npm install`

#### Build project
`npm run-script build`

#### Start server
`npm start`
